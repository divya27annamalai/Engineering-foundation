Course Overview

This course introduces the principles of computer networks. You'll learn about network architectures, OSI and TCP/IP models, physical and data link layers, network and transport layers, routing and switching, and network applications. You'll also get hands-on with socket programming and basic network security concepts.

Learning Outcomes

By the end of this course, you will be able to:

  • Explain the OSI and TCP/IP reference models and their layers.
  • Understand network topologies, media, and transmission methods.
  • Configure IP addressing and subnetting.
  • Explain routing protocols (RIP, OSPF, BGP) and switching.
  • Implement client-server communication using sockets.
  • Identify common network threats and security measures.
