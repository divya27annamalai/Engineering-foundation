Course Overview

This course covers the fundamentals of database management systems. You'll learn data modeling, ER diagrams, relational database design, SQL (DDL, DML, joins, subqueries), normalization, transaction management, and indexing. You'll work with MySQL and gain practical skills for managing data.

Learning Outcomes

By the end of this course, you will be able to:

  • Design and implement relational databases using ER modeling.
  • Write complex SQL queries for data retrieval and manipulation.
  • Normalize database schemas to reduce redundancy.
  • Understand ACID properties and transaction management.
  • Use indexing to optimize query performance.
  • Manage users, permissions, and backup/restore.
